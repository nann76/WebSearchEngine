公共计算机基础教学部

南开大学公共计算机基础教学部成立于1997年。2017年，教学部被认定为南开大学“优秀教学团队”，2020年被认定为南开大学“融合课程建设校级教学团队”。

公共计算机基础教学部始终秉承“通识教育是在高等教育阶段面向全体学生实施的综合素质教育，是一种非专业性、非工具性的基础教育”理念，高度重视教学理念、课程结构、课程内容的改革创新，不断改变传统以知识灌输为中心的教学方式，积极探索符合通识教育理念和本质的授课方式。教学部主任赵宏老师提出的基于OBE的教学模式，将“3A5S（三能五会）”作为大学生计算机通识素养的教学目标，即“能想，能用，能创新”；“会发现问题，会求解问题，会团队合作，会写论文，会展示”。该模式真正实现了从“以教师为中心”到“以学生为中心”的转变；真正实现了学生从“学会知识”到“具有能力”的转变。本团队的教学理念和教学改革成果受到各高校同行的认可和好评。教学部老师多次应邀在教育部教学指导委员会等各级全国教学研讨会上，介绍和推广团队的教学理念和教学方法，得到参会代表的得认可和好评，认为我们南开的计算机通识教育走在了全国的前列，具有启发作用。

2020年，教学部满足大类招生的需要，提出了新的计算机通识教育课程体系，未来可为南开大学本科生提供不同系列的套餐，包括二十余门课程。内容涵盖“物联网”、“人工智能”、“数据分析”、“数据可视化”、“机器学习”、“区块链”等热点。2021年，教学部配合学校“四新”专业发展和建设，提出了“分类分层次”的计算智能与“四新”专业融合课程体系，建设了“新文科智能计算基础”、“新理工科智能计算基础”等多门专业数据科融合课程，以配合一流大学、一流学科的“双一流”建设，也契合了目前“新基建”后备人才的培养目标。

在教学资源和教学平台建设上，本团队已实现全部必修课程MOOC化，在国内MOOC平台上线6门课程，在面向国际的MOOC平台上线5门课程。 目前所有教学班级全部使用自建MOOC/SPOC开展线上线下混合式教学。从学生的培养成果和评价反馈中可以看出，充分的线上资源和在线直播，结合线下同步混合式教学，圆满达成了“3A5S”教学目标，得到了学生的普遍认可和好评。

本团队在基础课教学的同时，也投入了很大精力进行教学和科学研究，取得多项研究成果，获得多项荣誉：    

  • 发表教学研究论文和科学研究论文百余篇；

  • 出版著作或教材二十余部，实现必修课程自编教材全覆盖；

  • 主持或参加国家级、省部级及校级教学改革和科学研究项目数十项；

  • 天津市级一流本科建设课程6项；

  • 南开大学教学成果一等奖2项，二等奖5项；

  • 南开大学教学创新大赛一等奖1项，二等奖1项；

  • 南开大学教学名师1名；

  • 南开大学教育教学奖优秀青年教师奖1名；

  • 南开大学唐立新基金优秀教学教师奖1名；

  • 南开大学课程思政优秀典型2名。

本团队还积极参与社会活动，勇于担当。团队承担了“中国大学生计算机设计大赛天津市赛区”的筹备和主办工作，至今已成功承办了九届。团队教师将课程教学与大赛紧密结合，指导学生获天津市级赛及全国总决赛各级奖项数十项，有效地培养了学生的创新意识和能力。
