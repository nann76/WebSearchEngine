计算机学院2022级学硕班主题团日活动
发布时间： 2022-11-20 发布人： 谭彤彤 浏览次数： 3

为深入学习、宣传贯彻党的二十大精神，南开大学计算机学院2022级学硕班开展了“学习二十大 运动展风采”主题团日活动，旨在筑牢支部成员思想根基，团结和引领班级学生用实际行动诠释青年担当，号召班级成员在实现中华民族伟大复兴中国梦的新征程上奋勇前进。

活动第一项为集体学习党的二十大报告精神，同学们交流分享自身的心得体会和切身感悟，强化支部成员政治品格，赓续红色血脉，夯实精神之本。计算机学院2022级学硕党支书王智弘从如何学习党的二十大报告、发挥党支部的战斗堡垒作用和传承南开体育精神三部分出发，带领团支部成员学习二十大报告精神，分享他对南开体育精神的理解和感悟，“我们应当牢记习近平总书记的嘱托，全面贯彻习近平新时代中国特色社会主义思想，以党的创新理论成果武装自己，自信自强，守正创新，强健体魄，为全面建设社会主义现代化国家、全面推进中华民族伟大复兴而团结奋斗。”

活动第二项为趣味运动。支部以丰富课余生活、强健体魄为目标，开展了躲避球和飞盘活动，以此传承张伯苓老校长“以体育人”的教育理念，发扬南开大学优良的体育传统。支部成员在运动中展现出昂扬向上的姿态和最美的青春风采。

通过本次团日活动，加深了支部成员对党的二十大报告精神内涵的理解，清晰认识到自身肩负的历史使命，增强了思想自觉和行动自觉，同时在体育运动中培养了支部成员不畏挫折、勇于拼搏、遵守规则、善于合作、珍惜荣誉、超越梦想的体育精神。

活动结束后，支部成员纷纷表示，作为新时代的青年，我们应当认真学习贯彻党的二十大精神，牢记习近平总书记的嘱托，不忘初心，笃力前行，担负起属于我们的时代重任，强健体魄，踔厉奋发，书写新时代的华丽篇章。
