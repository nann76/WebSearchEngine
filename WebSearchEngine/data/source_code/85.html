南开大学关于2022年秋季学期研究生教学工作预案
发布时间： 2022-06-27 发布人： 徐晓筠 浏览次数： 3

南开大学关于2022年秋季学期研究生教学工作预案

 

根据天津市教委以及学校疫情防控小组的工作要求，为进一步做好我校疫情防控常态化下研究生教学工作，保障师生身体健康，确保研究生教学质量，现制定以下工作预案：

一、开学与报到

    2022级新生报到时间为9月2日（星期五）；其他年级在校生报到时间为8月27日（星期六）-8月28日（星期日）。

研究生开学报到应于学校确定的开学日期的15日内完成，不得由他人代为报到，全校研究生按照确定的开学日期按时返校。因疫情隔离等原因不能按时返校报到的研究生，须按照要求办理延缓报到或休学手续。

二、授课安排

1.2021年秋季学期，在疫情稳定情况下原则上采取线下教学的方式，如遇部分学生因疫情原因不能返校参加线下学习的，各培养单位要做好线上线下教学同步开展预案；为保证研究生教学秩序，各培养单位在准备线下教学的同时，要做好开展线上教学的预案，根据疫情防控情况，可随时切换授课形式。具体教学安排视疫情情况进行调整，如有变化开学前将另行通知。

2.要加强课程考勤管理，做好教学过程中的健康状况检测。任课教师要调查了解学生的缺课缺勤原因，若因疾病或不适症状缺课缺勤，应详细记录所患疾病或症状信息，及时将相关情况通报开课单位教务部门，并由教务部门报告研究生院以及所在单位学生管理部门。

3.各培养单位应根据实际情况开展实习实践活动，可在校内完成的实习、实践类课程，按照学校疫情防控要求合理设计实施方案；需校外开展的实习、实践类课程，应根据学校疫情防控要求以及实习实践地区和单位要求，在保证学生安全的前提下进行。

三、研究生指导工作

研究生导师要充分发挥研究生培养第一责任人的作用，及时和全面掌握研究生的思想动态和身心状况，帮助研究生舒缓压力；要与研究生保持联络畅通，指导研究生阅读高水平专业文献，对研究生的学术进度、学术成果和学术水平严格把关，做好研究生的线上学习答疑与学位论文指导工作。导师须定期与所指导的研究生保持联系。

    四、学位授予及毕业前期工作

1.毕业资格审查

按照疫情防控要求，提前做好秋季学期毕业研究生的答辩资格审查工作以及学位授予条件审核工作。

2.论文送审

按照南开大学论文评阅管理规定执行，论文送审采取线上和线下送审相结合的方式。

3.答辩

毕业生答辩工作将视疫情发展情况，采取线上、线下答辩相结合的方式，确保符合条件的学生按期毕业和获取学位。各培养单位要做好延期毕业学生的指导工作，根据实际情况，有针对性地对延期毕业研究生进行指导，特别要关注临近最长修业年限研究生的学习和心理状况。

五、工作分工

1.研究生院：负责研究生整体教学安排和质量监控；

2.学院等培养单位：作为责任主体开展本单位教学组织管理、传达学校教学工作安排、教师和学生疫情防控宣传教育工作，协助本单位任课教师开展线上线下混合式教学，安排专人做好本单位教学场所的日常管理。

3.各单位分党委：对教师线上教学资源进行审核，确保意识形态安全。

4.任课教师：做好课堂教学、线上线下混合式教学准备工作，依据学校要求调整授课计划，学习和掌握学校疫情防控相关工作要求和应急处理流程，检测教学活动中的学生健康状况。

5.研究生：遵守学校疫情防控相关管理规定，坚持体育锻炼，科学佩戴口罩，注意与导师、辅导员以及学院教务保持联系。
