关于推荐优秀学生参与2022年天津市高校资助育人宣传大使选拔赛的工作通知
发布时间： 2022-11-30 发布人： 薛颖 浏览次数： 2

根据天津市教委相关文件要求，现将我校推荐优秀学生参与2022年天津市高校资助育人宣传大使选拔赛的工作做如下安排。

一、活动简介

为学习宣传贯彻党的二十大精神和天津市第十二次党代会精神，落实《市教委关于进一步加强新时代学生资助工作的通知》（津教政〔2021〕3号）精神，持续加大学生资助政策宣传力度，拓展宣传途径，天津市委教工委、市教委决定在全市高等学校开展“青春感党恩 筑梦新征程”2022年天津市高校资助育人宣传大使选拔赛。

二、报名条件

本次大赛可以学生个人参加，也可由2-3名学生组队参加。

参赛学生或学生团队成员均需满足以下条件：

1.我校全日制在籍在校学生；

2.在校期间表现优秀，顽强拼搏、自立自强、奋发成才，德智体美劳全面发展，无违法违纪行为；

3.在学期间曾受到各级各类学生资助政策奖励或资助；

4.认真研学习近平新时代中国特色社会主义思想，坚定不移贯彻执行党的路线、方针、政策，自觉践行社会主义核心价值观；

5.对于党和国家的学生资助政策及本校的资助体系有一定了解；宣传思路清晰，语言表达流畅，有感召力、亲和力。

三、报名方式

学校最终将推荐1名（队）同学参与市级推荐赛，请有意愿报名参加校级推荐选拔的同学，在12月1日17:00前填写附件1-《2022年天津市高校资助育人宣传大使选拔赛报名表》，将电子版文件发送至019041@nankai.edu.cn。
