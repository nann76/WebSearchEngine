【专业实践】关于做好2021级专业学位研究生专业实践工作的通知
发布时间： 2022-05-04 发布人： 徐晓筠 浏览次数： 1

转发研究生院专业学位办通知， 请同学们遵照执行。如需外出实习，需遵守学校疫情防控相关规定并填写附件的“校外实习登记表模板”报学院备案。

 

 

关于做好2021级专业学位研究生专业实践工作的通知

 

各培养单位：

专业实践是专业学位研究生培养的重要环节，是增强实践能力的重要保障。为做好2021级专业学位研究生专业实践工作，特发此通知。请各学院严格按照《南开大学硕士专业学位研究生专业实践管理规定》（南发字〔2017〕87 号）和《南开大学硕士专业学位研究生专业实践安全管理规定》（南发字〔2017〕88 号）文件规定，认真做好此项工作。

第一，认真做好实践前期准备。相关学院应成立专业实践工作小组，负责组织、协调和领导本单位专业学位研究生专业实践各项工作。学生在导师的指导下制定专业实践学习计划，填写《南开大学硕士专业学位研究生专业实践计划表》，经导师和学院领导签署意见盖章后, 报送学院存档备案。

第二，做好常态化疫情防控工作。各学院应根据学校疫情防控要求以及实习实践地区和单位要求，提前做好疫情防控预案，在保证学生安全的前提下组织学生进行专业实践。

第三，严格保证专业实践时间。专业学位研究生在学期间，必须保证不少于半年的专业实践，可采用集中实践与分段实践相结合的方式；生源为应届本科毕业生的专业实践时间原则上不少于 1 年。部分专业学位类别（领域）教指委有特殊规定的，按教指委规定执行。请各学院严格按照以上时间要求安排研究生的专业实践。

第四，扎实实施岗前教育培训。学院负责组织安全、保密和知识产权培训。参加专业实践的学生须认真阅读《南开大学硕士专业学位研究生专业实践安全管理规定》并接受安全、保密和知识产权等方面的教育，方可进入实践基地或实践单位学习。

第五，切实做到人身意外保险全覆盖。学生进入专业实践基地前必须由所在学院统一购买人身意外伤害保险，否则将不能被允许进入实践基地进行专业实践。学院应当根据学生的个人保险情况，积极同医疗保险办公室联系，实现人身意外伤害保险全覆盖。

第六，做好专业实践总结与鉴定工作。研究生应结合实践活动内容，认真总结自己在专业知识、职业素养、解决实际问题能力等方面的收获及体会。指导教师应积极指导研究生专业实践，全面客观评议研究生的实践表现及实践效果，实事求是给出鉴定意见。

第七，认真做好专业实践考核工作。专业实践时间不足或专业实践鉴定成绩为不合格的学生，专业实践环节不记学分，不得进入学位论文阶段。

 

 

研究生院专业学位办公室

2022年5月4日
