海河青听 | “我为学生做件事”——科研方向任你选，导师引领促成长
发布时间： 2021-05-28 发布人： 王怡然 浏览次数： 64402

      2021年5月27日下午14:30，计算机学院为同学们组织了“卓越班科研导师双选-走进实验室”活动，这是继2019年12月的“学院实验室科研信息发布会”活动后，学院又一次向大家介绍各个实验室的科研信息，旨在向大家介绍学院现有的科研方向和科研资源，帮助同学们破除迷茫，引导大家尽早树立明确的学术目标。

      会议以线上线下相结合的模式进行，线下部分在大通学活203教室进行，学院党委书记黄家友，数据与智能安全实验室的刘哲理老师，南开百度联合实验室的王刚、李雨森老师，计算机网络与信息安全研究室的蒲凌君老师，数据库与信息系统研究室的张莹、宋春瑶老师，生物信息与智能医学中心的刘健老师，嵌入式系统与信息安全实验室的宫晓利老师，软件安全与测试研究室的范玲玲老师，信息安全新技术研究室的王志老师，密码学与系统安全实验室的汪定老师，媒体计算与计算机视觉实验室的程明明、杨巨峰老师，以及大一到大三年级的计算机科学与技术卓越班学生参加了线下会议；学院其他同学通过飞书线上实时参与了此次会议。

      会上，各实验室的相关负责教师生动地向大家展示了实验室各自的课题方向、科研活动及文化环境建设情况，此外，杨巨峰老师还与大家分享了个人关于本科生参与科研训练的一些建议和心得。参会的同学们此次收获颇丰；最后，张莹老师向大家介绍了学院对“卓越班”的“科研导师”建设模式及“培养计算机领域领军人才”的目标，希望大家珍惜并充分利用学院的科研资源，早日走上时候自己的发展道路。

      据悉，卓越班同学“科研导师双选”机制将纳入学院后台管理系统，同学们可以登录学院系统后台完成选题、选导师的相关程序。
